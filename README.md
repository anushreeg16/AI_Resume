# ResumeAI - Smart Resume Screening & Candidate Ranking Tool

ResumeAI is an AI-powered resume screening and candidate ranking system designed to automate the recruitment process. The application analyzes uploaded resumes, extracts relevant information, identifies skills, matches them with job requirements, and generates an ATS-based compatibility score.

The system helps recruiters quickly evaluate multiple candidates by ranking resumes based on skills, experience, and job relevance. It reduces manual screening time and provides a smarter, faster, and more efficient hiring approach.

## Key Features
- 📄 Resume PDF upload and text extraction
- 🤖 AI-based resume analysis
- 📊 ATS compatibility score generation
- 🔍 Skill and keyword matching
- 🏆 Candidate ranking based on job requirements
- ⚡ Automated resume screening process
- 📋 Detailed analysis report generation

## Technologies Used
- Python
- Flask Framework
- HTML, CSS, JavaScript
- pdfplumber for PDF text extraction
- AI-based scoring and matching algorithms

## Objective
The main objective of ResumeAI is to build an intelligent recruitment assistant that helps HR professionals identify suitable candidates efficiently by analyzing resumes automatically and reducing the effort required for manual screening.
